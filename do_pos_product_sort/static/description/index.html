POS Product Sort

Key Features:

We Have added various Sort functionalities in Point of Sale, allowing users to sort and view products by their sales volume, new arrivals, price in ascending or descending order, and name in ascending or descending order, as well as Sort by the least sold items.

  • Sort by Most Sold.
  • Sort by New Arrivals.
  • Sort by Price ASC.
  • Sort by Price DESC.
  • Sort by Name ASC.
  • Sort by Name DESC.
  • Sort by Least Sold.
